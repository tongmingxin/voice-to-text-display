# 语音转大字 - 听障沟通辅助工具

长按说话，语音识别后自动横屏全屏大字显示，方便与听障人士面对面沟通。

## 功能特点

- **长按说话**：按住按钮说话，松开后自动识别
- **大字显示**：识别完成后自动横屏全屏，超大字体清晰可读
- **持续对话**：横屏模式下按钮常驻，可反复识别多句话
- **双语支持**：支持普通话和粤语切换
- **跨平台**：浏览器运行，Android / iOS / 电脑均可使用

## 使用方法

1. 用手机浏览器打开 `index.html`（或部署后的网址）
2. 选择语言（普通话/粤语）
3. **长按**屏幕中央的红色按钮，对准手机说话
4. **松开**按钮，识别完成后自动切换到横屏大字展示
5. 将手机横屏递给对方查看
6. 需要继续说话时，长按底部的"长按说话"按钮
7. 点击"退出"回到竖屏模式，点击"清空"清除已有文字

## 浏览器要求

| 平台 | 推荐浏览器 |
|------|-----------|
| Android | Chrome |
| iOS (iPhone/iPad) | Safari |
| 电脑 | Chrome |

> 需要联网使用（语音识别依赖云端服务）

## 部署方式

### 方式一：本地直接使用
直接用浏览器打开 `index.html` 即可。

### 方式二：部署到服务器
将所有文件上传至任意静态服务器（如 GitHub Pages、Vercel、Netlify 等）。
使用 HTTPS 才能正常调用麦克风权限。

### 方式三：添加到主屏幕（推荐）
在手机浏览器中打开网页后，选择"添加到主屏幕"，即可像 App 一样使用。

## 技术栈

- HTML5 + CSS3 + JavaScript（纯前端，无需后端）
- Web Speech API（语音识别）
- Fullscreen API（全屏）
- Screen Orientation API（横屏锁定）
